Focusing on hosting solutions and demo of GKE

Hosting Solutions

  • Amazon Elastic Kubernetes Service
  • Azure Container Service
  • Google Kubernetes Engine

Amazon Elastic Kuberenetes Service

Advantages

  • Extremely well-written documentation and large community
  • Other services from AWS that can integrate well with EKS

Disadvantages

  • EKS configuration is not easy to setup
  • Slow build time, taking multiple minutes to create nodes
  • Subpar interface
  • Considerably costly
    • You will pay for master nodes, 20 cents/hr
    • 750 hours in a month will cost 150$ just for nodes

Azure Container Service

Advantages

  • Good interface
  • Allows for analytics tools provided by Microsoft
  • Free, just pay for resources for clusters
  • Most interactions will happen on Windows (disadvantage to me)

Disadvantages

  • Hard to use CLI
    • Force to used it more than the other hosting solutions
  • Smaller community than AWS and Google

My favorite one of all

Google Kubernetes Engine

Advantages

  • Best thing since sliced bread, greatest interface of the bunch
    • Extreme control over all components, great for dev-ops
    • Can edit config, modify workloads, services, types, and more
  • Deployments in just 2 clicks
  • Clusters take only a few minutes to create
  • $0.19 an hour, offers substantial discount for continued use

Disadvantages

  • Other services from Google aren't as great as AWS
  • Documentation is not great
  • Support isn't as great either
